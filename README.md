## 🔧 Easy Customization:

### **Colors:**
Edit the CSS variables at the top of `styles.css`:
```css
:root {
    --primary-color: #3498db;    /* Main blue color */
    --secondary-color: #2c3e50;  /* Dark text color */
    --accent-color: #2ecc71;     /* Success green */
}
```

### **Content:**
- **Personal Info:** Edit name, title, email in `index.html`
- **About Text:** Update the about section paragraphs
- **Experience:** Modify timeline items with your work history
- **Projects:** Replace with your own projects and links
- **Skills:** Update skill percentages and categories
- **Education:** Add your educational background

### **Images:**
- **Profile Photo:** Replace `images/profile.jpg` with your photo
- **Favicon:** Replace `images/favicon.png` with your icon

### **Social Links:**
Update the social media URLs in the contact section:
```html
<a href="https://www.linkedin.com/in/yourusername/" class="social-link" target="_blank">
<a href="https://github.com/yourusername" class="social-link" target="_blank">
<a href="mailto:your.email@domain.com" class="social-link">
```
